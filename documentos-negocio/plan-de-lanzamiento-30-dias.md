# Plan de Lanzamiento — 30 Días

Plan de acción concreto para pasar de cero a tus primeros clientes en un mes.
Marca cada tarea a medida que avanzas.

---

## Semana 1 — Montaje y demo

**Objetivo:** tener tu agencia online y una demo funcionando.

- [ ] Elige UN nicho para arrancar (recomendado: restaurantes o cerrajeros/oficios).
- [ ] Define el nombre y dominio de tu agencia.
- [ ] Publica la web de tu agencia (`web-agencia/`) en Netlify o Vercel (gratis).
- [ ] Crea cuentas: Vapi, Twilio, OpenAI (y ElevenLabs si quieres mejor voz).
- [ ] Monta el agente de voz demo siguiendo `agente-voz-ia/guia-configuracion.md`.
- [ ] Haz tu primera llamada de prueba a la IA.

## Semana 2 — Integración y demo de nicho

**Objetivo:** flujo completo de punta a punta funcionando.

- [ ] Despliega el backend (`backend-integracion/`) en Render o Railway.
- [ ] Conecta la herramienta `registrar_pedido` de Vapi con tu backend.
- [ ] Configura las notificaciones (Telegram o WhatsApp/Twilio).
- [ ] Prueba: llama → pedido → aparece en el dashboard → llega el aviso.
- [ ] Personaliza la web demo de nicho (`web-demo-restaurante/`) y publícala.

## Semana 3 — Prospección intensiva

**Objetivo:** llenar tu pipeline de prospectos.

- [ ] Arma una lista de 40-60 negocios del nicho (Google Maps, Instagram).
- [ ] Ejecuta la "Demo en Vivo" (ver `scripts-prospeccion.md`): 20 demos personalizadas.
- [ ] Ejecuta la "Auditoría de Llamada Perdida" con otros 20.
- [ ] Agenda al menos 5 llamadas de venta.
- [ ] Prepara tu guion de venta y practícalo.

## Semana 4 — Cierre y entrega

**Objetivo:** cerrar tus primeros 1-3 clientes.

- [ ] Realiza las llamadas de venta agendadas.
- [ ] Cierra al menos 1 cliente (aunque sea a precio promo por testimonio).
- [ ] Entrega: web + IA + integración funcionando.
- [ ] Pide testimonio en video y datos de resultados.
- [ ] Usa ese caso de éxito para cerrar al siguiente (sube el precio).

---

## Métricas que debes mirar
- N° de demos enviadas por día.
- Tasa de respuesta a tus mensajes.
- N° de llamadas de venta agendadas.
- N° de clientes cerrados.
- MRR acumulado (suma de mensualidades).

## Meta de los primeros 90 días
- **Mes 1:** 1-3 clientes, sistema validado.
- **Mes 2:** 4-6 clientes, primer caso de éxito documentado.
- **Mes 3:** 8-10 clientes → ~$1.500.000+/mes de ingreso recurrente.

> Una vez que tengas el sistema y un caso de éxito, todo se acelera: replicas
> la misma plantilla web y la misma config de IA cambiando solo los datos.
